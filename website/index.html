Overview

This guide provides all necessary information to help you understand, set up, and deploy the project efficiently.

This project is a DevOps initiative that incorporates various DevOps tools and is deployed in the cloud.

The primary objective of this project is to host a Linux cloud server, run Docker containers within it, and automate the configuration process using reliable and scalable tools.
